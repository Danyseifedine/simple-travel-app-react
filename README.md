# React Item Management Application

This React application allows users to add, check, adjust quantity, and filter items. It also tracks and displays the percentage of items picked up.

## Features
- Add Items: Easily add new items to the list.
- Check Items: Mark items as checked or unchecked.
- Adjust Quantity: Increase or decrease the quantity of each item.
- Filter Items: Filter items based on various criteria.
- Percentage Display: View the percentage of items that have been picked up.
